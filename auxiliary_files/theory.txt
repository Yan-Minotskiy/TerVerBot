*–¢–µ–æ—Ä–∏—è üìö*
==================
*–ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å*

–ü—Ä–∏ *–∫–ª–∞—Å—Å–∏—á–µ—Å–∫–æ–º –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏* –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ–º –† (A) = m/n, –≥–¥–µ m ‚Äî—á–∏—Å–ª–æ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –∏—Å—Ö–æ–¥–æ–≤ –∏—Å–ø—ã—Ç–∞–Ω–∏—è, –±–ª–∞–≥–æ–ø—Ä–∏—è—Ç—Å—Ç–≤—É—é—â–∏—Ö –ø–æ—è–≤–ª–µ–Ω–∏—é —Å–æ–±—ã—Ç–∏—è –ê; n ‚Äî–æ–±—â–µ–µ —á–∏—Å–ª–æ –≤–æ–∑–º–æ–∂–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã—Ö –∏—Å—Ö–æ–¥–æ–≤ –∏—Å–ø—ã—Ç–∞–Ω–∏—è. –ü—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç—Å—è, —á—Ç–æ —ç–ª–µ–º–µ–Ω—Ç–∞—Ä–Ω—ã–µ –∏—Å—Ö–æ–¥—ã –æ–±—Ä–∞–∑—É—é—Ç –ø–æ–ª–Ω—É—é –≥—Ä—É–ø–ø—É –∏ —Ä–∞–≤–Ω–æ–≤–æ–∑–º–æ–∂–Ω—ã. *–û—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–∞—è —á–∞—Å—Ç–æ—Ç–∞* —Å–æ–±—ã—Ç–∏—è –ê –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ–ºW (A) = m/n, –≥–¥–µ m ‚Äî —á–∏—Å–ª–æ –∏—Å–ø—ã—Ç–∞–Ω–∏–π, –≤ –∫–æ—Ç–æ—Ä—ã—Ö —Å–æ–±—ã—Ç–∏–µ –ê –Ω–∞—Å—Ç—É–ø–∏–ª–æ; n ‚Äî –æ–±—â–µ–µ —á–∏—Å–ª–æ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–Ω—ã—Ö –∏—Å–ø—ã—Ç–∞–Ω–∏–π.
–í–≤–µ–¥–∏—Ç–µ /simple, —á—Ç–æ–±—ã –ø–æ—Å—á–∏—Ç–∞—Ç—å –∫–ª–∞—Å—Å–∏—á–µ—Å–∫—É—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å.%
*–ö–æ–º–±–∏–Ω–∞—Ç–æ—Ä–∏–∫–∞*

–î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –≤ –∑–∞–¥–∞—á–∞—Ö –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∏–Ω–æ–≥–¥–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∑–Ω–∞—Ç—å —á–∏—Å–ª–æ –±–ª–∞–≥–æ–ø—Ä–∏—è—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π –∏ –æ–±—â–µ–µ —á–∏—Å–ª–æ —Å–æ–±—ã—Ç–∏–π, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–Ω–æ –Ω–∞–π—Ç–∏ —Å –ø–æ–º–æ—â—å—é —Ñ–æ—Ä–º—É–ª –∫–æ–º–±–∏–Ω–∞—Ç–æ—Ä–∏–∫–∏. –î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –ø–µ—Ä–µ–π—Ç–∏ –≤ —Ä–∞–∑–¥–µ–ª –∫–æ–º–±–∏–Ω–∞—Ç–æ—Ä–∏–∫–∏ –≤–≤–µ–¥–∏—Ç–µ /combinatoric.
*–ß–∏—Å–ª–æ –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–æ–∫* –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –ø–æ —Ñ–æ—Ä–º—É–ª–µ:
`Pn = n!`
/permutation - —á—Ç–æ–±—ã —Ä–∞—Å—Å—á–∏—Ç–∞—Ç—å
*–ß–∏—Å–ª–æ —Ä–∞–∑–º–µ—â–µ–Ω–∏–π:*
`Ank = n! / (n - k)! = Pn / (n - k)!`
/placement - —á—Ç–æ–±—ã —Ä–∞—Å—Å—á–∏—Ç–∞—Ç—å
*–ß–∏—Å–ª–æ —Å–æ—á–µ—Ç–∞–Ω–∏–π:*
`Cnk = n! / (n-k)!k! = Ank / k!`
/combinations - —á—Ç–æ–±—ã —Ä–∞—Å—Å—á–∏—Ç–∞—Ç—å%
*–¢–µ–æ—Ä–µ–º—ã —Å–ª–æ–∂–µ–Ω–∏—è –∏ —É–º–Ω–æ–∂–µ–Ω–∏—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π*

*üìå–¢–µ–æ—Ä–µ–º–∞ —Å–ª–æ–∂–µ–Ω–∏—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π –Ω–µ—Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π.* –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ—è–≤–ª–µ–Ω–∏—è –æ–¥–Ω–æ–≥–æ –∏–∑ –¥–≤—É—Ö –Ω–µ—Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π, –±–µ–∑—Ä–∞–∑–ª–∏—á–Ω–æ –∫–∞–∫–æ–≥–æ, —Ä–∞–≤–Ω–∞ —Å—É–º–º–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π —ç—Ç–∏—Ö —Å–æ–±—ã—Ç–∏–π:
`–† (–ê + –í) = –† (–ê) + –† (–í)`

*üìå–¢–µ–æ—Ä–µ–º–∞ —Å–ª–æ–∂–µ–Ω–∏—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π —Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π.* –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ—è–≤–ª–µ–Ω–∏—è —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ–≥–æ –∏–∑ –¥–≤—É—Ö —Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π —Ä–∞–≤–Ω–∞ —Å—É–º–º–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π —ç—Ç–∏—Ö —Å–æ–±—ã—Ç–∏–π –±–µ–∑ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –∏—Ö —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –ø–æ—è–≤–ª–µ–Ω–∏—è:
`–† (A + –í) = –† (A) + –† (–í) ‚Äî –†(–ê–í)`

*üìå–¢–µ–æ—Ä–µ–º–∞ —É–º–Ω–æ–∂–µ–Ω–∏—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π.* –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –ø–æ—è–≤–ª–µ–Ω–∏—è –¥–≤—É—Ö —Å–æ–±—ã—Ç–∏–π —Ä–∞–≤–Ω–∞ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –æ–¥–Ω–æ–≥–æ –∏–∑ –Ω–∏—Ö –Ω–∞ —É—Å–ª–æ–≤–Ω—É—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –¥—Ä—É–≥–æ–≥–æ, –≤—ã—á–∏—Å–ª–µ–Ω–Ω—É—é –≤ –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏–∏, —á—Ç–æ –ø–µ—Ä–≤–æ–µ —Å–æ–±—ã—Ç–∏–µ —É–∂–µ –Ω–∞—Å—Ç—É–ø–∏–ª–æ:
`–†(–ê–í) = –†(–ê)‚ãÖ–†a(–í)`
–í —á–∞—Å—Ç–Ω–æ—Å—Ç–∏, –¥–ª—è –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö —Å–æ–±—ã—Ç–∏–π
`–†(–ê–í) = –† (–ê)‚ãÖ–†(–í)`
—Ç. –µ. –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –ø–æ—è–≤–ª–µ–Ω–∏—è –¥–≤—É—Ö –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö —Å–æ–±—ã—Ç–∏–π —Ä–∞–≤–Ω–∞ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π —ç—Ç–∏—Ö —Å–æ–±—ã—Ç–∏–π.

–ï—Å–ª–∏ —Å–æ–±—ã—Ç–∏–µ –ê –º–æ–∂–µ—Ç –ø—Ä–æ–∏–∑–æ–π—Ç–∏ —Ç–æ–ª—å–∫–æ –ø—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ –æ–¥–Ω–æ–≥–æ –∏–∑ —Å–æ–±—ã—Ç–∏–π _B1, B2, ... Bn_, –∫–æ—Ç–æ—Ä—ã–µ –æ–±—Ä–∞–∑—É—é—Ç –ø–æ–ª–Ω—É—é –≥—Ä—É–ø–ø—É –Ω–µ—Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è –ê –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –ø–æ —Ñ–æ—Ä–º—É–ª–µ
`P(A) = P(B1)P(A|B1) + P(B2)P(A|B2) + ... + P(Bn)P(A|Bn)`
–≠—Ç–∞ —Ñ–æ—Ä–º—É–ª–∞ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è *—Ñ–æ—Ä–º—É–ª–æ–π –ø–æ–ª–Ω–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏*.
–ü—É—Å—Ç—å —Å–æ–±—ã—Ç–∏–µ –ê –º–æ–∂–µ—Ç –Ω–∞—Å—Ç—É–ø–∏—Ç—å –ª–∏—à—å –ø—Ä–∏ —É—Å–ª–æ–≤–∏–∏ –ø–æ—è–≤–ª–µ–Ω–∏—è –æ–¥–Ω–æ–≥–æ –∏–∑ –Ω–µ—Å–æ–≤–º–µ—Å—Ç–Ω—ã—Ö —Å–æ–±—ã—Ç–∏–π (–≥–∏–ø–æ—Ç–µ–∑) –í1 –∏ –í2, .... –ín, –∫–æ—Ç–æ—Ä—ã–µ –æ–±—Ä–∞–∑—É—é—Ç –ø–æ–ª–Ω—É—é –≥—Ä—É–ø–ø—É —Å–æ–±—ã—Ç–∏–π. –ï—Å–ª–∏ —Å–æ–±—ã—Ç–∏–µ –ê —É–∂–µ –ø—Ä–æ–∏–∑–æ—à–ª–æ, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –≥–∏–ø–æ—Ç–µ–∑ –º–æ–≥—É—Ç –±—ã—Ç—å –ø–µ—Ä–µ–æ—Ü–µ–Ω–µ–Ω—ã –ø–æ *—Ñ–æ—Ä–º—É–ª–∞–º –ë–µ–π–µ—Å–∞*:
`Pa(Bi) = P(Bi)‚ãÖPbi(A) / P(A)`%
*–ü–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ –∏—Å–ø—ã—Ç–∞–Ω–∏–π*

–ï—Å–ª–∏ –ø—Ä–æ–∏–∑–≤–æ–¥—è—Ç—Å—è –∏—Å–ø—ã—Ç–∞–Ω–∏—è, –ø—Ä–∏ –∫–æ—Ç–æ—Ä—ã—Ö –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ—è–≤–ª–µ–Ω–∏—è —Å–æ–±—ã—Ç–∏—è –ê –≤ –∫–∞–∂–¥–æ–º –∏—Å–ø—ã—Ç–∞–Ω–∏–∏ –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç –∏—Å—Ö–æ–¥–æ–≤ –¥—Ä—É–≥–∏—Ö
–∏—Å–ø—ã—Ç–∞–Ω–∏–π, —Ç–æ —Ç–∞–∫–∏–µ –∏—Å–ø—ã—Ç–∞–Ω–∏—è –Ω–∞–∑—ã–≤–∞—é—Ç *–Ω–µ–∑–∞–≤–∏—Å–∏–º—ã–º–∏ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ —Å–æ–±—ã—Ç–∏—è* A.

*–§–æ—Ä–º—É–ª–∞ –ë–µ—Ä–Ω—É–ª–ª–∏.* –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ –≤ –ø –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö –∏—Å–ø—ã—Ç–∞–Ω–∏—è—Ö, –≤ –∫–∞–∂–¥–æ–º –∏–∑ –∫–æ—Ç–æ—Ä—ã—Ö –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ—è–≤–ª–µ–Ω–∏—è —Å–æ–±—ã—Ç–∏—è —Ä–∞–≤–Ω–∞ —Ä (0 < —Ä < 1), —Å–æ–±—ã—Ç–∏–µ –Ω–∞—Å—Ç—É–ø–∏—Ç —Ä–æ–≤–Ω–æ k —Ä–∞–∑ (–±–µ–∑—Ä–∞–∑–ª–∏—á–Ω–æ, –≤ –∫–∞–∫–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏), —Ä–∞–≤–Ω–∞:
`P = C(k, n)‚ãÖp^k‚ãÖq^(n-k)`

–ï—Å–ª–∏ —á–∏—Å–ª–æ –∏—Å–ø—ã—Ç–∞–Ω–∏–π n –≤–µ–ª–∏–∫–æ, –∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ä –Ω–µ –æ—á–µ–Ω—å –º–∞–ª–∞, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —Å–æ–±—ã—Ç–∏–µ –ê –Ω–∞—Å—Ç—É–ø–∏—Ç —Ä–æ–≤–Ω–æ k —Ä–∞–∑ –Ω–∞—Ö–æ–¥—è—Ç –ø–æ *–ª–æ–∫–∞–ª—å–Ω–æ–π —Ñ–æ—Ä–º—É–ª–µ –õ–∞–ø–ª–∞—Å–∞* (–ú—É–∞–≤—Ä–∞-–õ–∞–ø–ª–∞—Å–∞) —Å–º. –∫–∞—Ä—Ç–∏–Ω–∫—Éüëá –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —Å–æ–±—ã—Ç–∏–µ –ê –ø–æ—è–≤–∏—Ç—Å—è –≤ n –∏—Å–ø—ã—Ç–∞–Ω–∏—è—Ö –Ω–µ –º–µ–Ω–µ–µ k1 –∏ –Ω–µ –±–æ–ª–µ–µ k2 —Ä–∞–∑ –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –ø–æ *–∏–Ω—Ç–µ–≥—Ä–∞–ª—å–Ω–æ–π —Ñ–æ—Ä–º—É–ª–µ –õ–∞–ø–ª–∞—Å–∞*:
`–†n(k1 ‚â§ k ‚â§ k2) = Œ¶(x2) ‚àí Œ¶(x1)`
–ó–Ω–∞—á–µ–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–π –õ–∞–ø–ª–∞—Å–∞ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –∏–∑ [—Ç–∞–±–ª–∏—Ü—ã](http://www.ekonomika-st.ru/drugie/metodi/t-ver-1-4-0.html)

–ï—Å–ª–∏ —á–∏—Å–ª–æ –∏—Å–ø—ã—Ç–∞–Ω–∏–π n –≤–µ–ª–∏–∫–æ, –∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å p –ø–æ—è–≤–ª–µ–Ω–∏—è —Å–æ–±—ã—Ç–∏—è –ê –≤ –∫–∞–∂–¥–æ–º –∏—Å–ø—ã—Ç–∞–Ω–∏–∏ –æ—á–µ–Ω—å –º–∞–ª–∞, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ —ç—Ç–æ —Å–æ–±—ã—Ç–∏–µ –Ω–∞—Å—Ç—É–ø–∏—Ç —Ä–æ–≤–Ω–æ k —Ä–∞–∑ –Ω–∞—Ö–æ–¥—è—Ç –ø–æ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–Ω–æ–π *—Ñ–æ—Ä–º—É–ª–µ –ü—É–∞—Å—Å–æ–Ω–∞* —Å–º. –∫–∞—Ä—Ç–∏–Ω–∫—Éüëá%
